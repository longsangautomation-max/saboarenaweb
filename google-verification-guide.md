URL: https://search.google.com/search-console

## 2. Select Property:
- Choose: sc-domain:saboarena.com

## 3. Check URL Inspection:
- Go to URL Inspection tool
- Test any URL from our list:
  * https://saboarena.com/user/30894dda-74f7-4e95-8749-65a098778901
  * https://saboarena.com/live-match/7c65e203-99d2-4dc8-9c43-a781ce3fd110

## 4. Check Coverage Report:
- Go to Coverage in left menu
- Look for 'Valid' pages count
- Should increase from 7 to 200+ over next few days

## 5. Check Performance:
- Go to Performance tab
- Monitor impressions and clicks
- Filter by pages containing '/user/' or '/live-match/'

## 6. Check Sitemaps:
- Go to Sitemaps section
- Submit: https://saboarena.com/sitemap.xml
- Submit: https://saboarena.com/sitemap-index.xml
